# HUSTOJ-Ballon-System

稍微修改一下比赛地址账号ID啥的就可以连通了

等待三秒会列出待送的前十个气球，每一个气球都有一个唯一的编号

格式为：

&emsp;气球编号  选手ID  通过的题目

功能：

&emsp;输入某个气球的编号删掉这个气球，并刷新列表

&emsp;输入0主动刷新列表

&emsp;输入-1会查看已送的气球列表

&emsp;当有新气球产生是会发出提醒
